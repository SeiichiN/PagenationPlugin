# PagenationPlugin
Simple Pagenation for WordPress

## 使い方
ページネーションを使いたいところに以下のタグを記入するだけ。

```php:タグ
<?php billies_pagenation(); ?>
```
 
**（注）** ただし、WordPressのメインループの外でなければならない。
